CineRent - Sistema de Gestión de Alquileres

Este proyecto es una aplicación web desarrollada con Spring Boot y Thymeleaf para gestionar una tienda de alquiler de películas, utilizando la base de datos de ejemplo Pagila (PostgreSQL).

## Funcionalidades Principales
* **Dashboard:** Indicadores clave de negocio y gráficas de ingresos.
* **Catálogo:** Buscador avanzado con filtros por Título, Categoría, Actor y Clasificación (Rating).
* **Clientes:** Gestión de clientes con historial detallado de rentas.
* **Rentas:** Consulta de movimientos por rango de fechas y tienda.
* **Reportes:** Rankings de películas más vistas y mejores clientes.

## Tecnologías Utilizadas
* Java 17
* Spring Boot 3.x
* Spring Data JPA
* Thymeleaf
* PostgreSQL (Base de datos Pagila)
* Maven

## Requisitos Previos
1. Tener instalado **Java JDK 17**.
2. Tener instalado **PostgreSQL**.
3. Tener cargada la base de datos **Pagila**.
